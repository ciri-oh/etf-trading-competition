## 背景

韩国股市（尤其是半导体板块）与 A 股半导体板块高度联动。三星电子、SK海力士是全球半导体风向标，其涨跌直接影响 A 股芯片/半导体 ETF 的市场情绪和走势。

## 分析内容（所有评估节点必做）

### 1. KOSPI 指数午盘走势
- 查询 KOSPI 指数午盘涨跌幅
- 判断韩国市场整体情绪（偏多/偏空/震荡）
- 与 A 股主要指数对比（是否同步）

### 2. 韩国半导体龙头
- **三星电子**（005930.KS）— 全球存储芯片龙头
- **SK海力士**（000660.KS）— 全球第二大存储芯片厂商
- 关注：涨跌幅、成交量异动、重大新闻

### 3. 联动关系分析
- 韩国半导体下跌 → A 股半导体 ETF 承压（情绪传导）
- 韩国半导体反弹 → A 股半导体可能跟涨
- 韩国半导体独立走势 → A 股半导体可能有独立逻辑

### 4. 午间重要新闻
- 韩国产业政策（半导体补贴、出口管制）
- 全球存储芯片价格动态
- 韩元汇率波动（影响外资流向）

## 极端波动模式（2026-07-06 新增）

### 三星/海力士业绩"利好出尽"暴跌模式（2026-07-07 新增）

**现象：** 2026-07-07 三星电子公布Q2初步业绩：营业利润89.4万亿韩元(+1810% YoY)，营收171万亿韩元(+129% YoY)，均超市场预期。但开盘暴跌超6%，SK海力士一度跌3.8%，KOSPI一度跌超4%。

**根因分析：**
- 业绩预期已被充分定价（三星年内涨超125%），属于经典的"买预期，卖事实"
- 摩根士丹利报告指出半导体板块动能衰竭，资金正转向超算巨头（微软、亚马逊、Meta）
- 韩国杠杆ETF（2倍杠杆）的每日调仓机制加剧了下跌抛压
- 韩国股市结构性失衡（三星+SK海力士占KOSPI权重>43%），单一股票暴跌即拖累全指数

**判断框架：**
- 季度业绩发布日（三星通常在月初、海力士在月中）是高风险事件日
- 业绩超预期但股价暴跌 → 利好出尽模式，通常持续1-3个交易日
- 业绩低于预期但股价上涨 → 利空出尽模式，可能是左侧建仓机会
- 关键区分：业绩是否已被"price in"（看业绩公布前1个月的涨幅）

**对A股半导体影响评估：**
- 韩国半导体利好出尽暴跌对A股半导体有情绪压制，但A股可能展现独立走势
- 如果A股半导体已提前经历调整（如7/2-7/3累计跌超10%），则"免疫效应"更强
- 关键判断指标：A股半导体ETF跌幅是否小于KOSPI跌幅的1/3？若是 → 脱钩成立
- 验证案例（2026-07-07）：三星-6%/KOSPI-4% vs 159516+1.63%/512760+0.35% → A股半导体逆势翻红，完全脱钩

**输出示例：**
```
韩国半导体今日"利好出尽"暴跌（三星Q2业绩创纪录但跌6%），
但A股半导体ETF逆势翻红（159516+1.63%, 512760+0.35%），
验证了WSTS/SEAJ行业利好提供了独立于韩股的支撑力，
A股半导体已提前调整，对外盘波动的敏感度降低。
```

## 极端波动模式（2026-07-06 新增）

### 韩国半导体与A股半导体脱钩模式

**现象：** 2026-07-06 KOSPI 从开盘+5.78%一路暴跌至-3%（全天振幅近9%），SK海力士-5%、三星-1.6%。但 A 股半导体 ETF（512760 +0.43%）仅微跌，与韩股大幅脱钩。

**判断框架：**
- A 股半导体已提前经历上周大跌调整（7/2-7/3 累计跌超10%），短期超卖修复中
- 韩国半导体暴跌对 A 股半导体的情绪传导正在减弱（"免疫效应"：第一次暴跌冲击大，第二次冲击小）
- 关键判断指标：A 股半导体 ETF 跌幅是否小于 KOSPI 跌幅的 1/3？若是 → 脱钩成立

**输出示例：**
```
韩国半导体尾盘暴跌(-3%)，但A股半导体今日表现明显强于韩股
(512760+0.43% vs SK海力士-5%)，短期超卖修复中，对外盘波动的敏感度正在降低。
```

### KOSPI 极端过山车日（振幅 > 8%）

**特征：** KOSPI 单日振幅超过 8%（如 7/6 从+5.78%到-3%），通常伴随外资巨额进出和散户杠杆交易。

**评估要点：**
- 韩国股市结构性失衡（三星+SK海力士占 KOSPI 权重 > 43%）
- 外资创纪录净卖出 + 散户加杠杆接盘 → 市场根基脆弱
- 极端波动日后的次日通常有惯性延续（高开或低开）
- 关注今晚美股费城半导体指数走势，决定明日 A 股半导体方向
