## 一、逆向投资框架

### 核心原则
- **别人恐惧时贪婪，别人贪婪时恐惧**
- 超买超卖信号：RSI > 70 超买区，RSI < 30 超卖区
- 恐慌贪婪指数：VIX飙升+放量下跌 = 极度恐惧，准备左侧建仓
- 恐慌贪婪指数：VIX低位+缩量上涨 = 过度贪婪，准备减仓

### 适用场景
- 市场极端波动时（大跌/大涨）
- 单只ETF连续涨跌3天以上
- 成交量异常放大/缩小

### 操作指引
| 信号 | 动作 | 仓位 |
|------|------|------|
| RSI<30 + 放量下跌 | 左侧试探买入 | 5-10% |
| RSI<20 + 恐慌情绪 | 逐步加仓 | 10-15% |
| RSI>70 + 缩量上涨 | 分批止盈 | 减仓1/3 |
| RSI>80 + 人人喊涨 | 全面减仓 | 降至30%以下 |

### 价格突变处理
- 单日涨幅超过3%后，不宜追高，等回踩支撑位
- 若从晨间策略目标价1.88涨至1.976（涨幅5.11%），应视为错过买点，放弃本次建仓，不追高
- 追高是常见散户错误，见「六、市场心理框架」

## 二、宏观分析框架

### 美林时钟四象限
| 阶段 | 经济 | 通胀 | 推荐板块 |
|------|------|------|----------|
| 复苏期 | ↑ | ↓ | 成长（半导体、AI、新能源） |
| 过热期 | ↑ | ↑ | 周期（有色、煤炭、化工） |
| 滞胀期 | ↓ | ↑ | 防御（银行、红利、消费） |
| 衰退期 | ↓ | ↓ | 债券（公司债ETF、国债ETF） |

### 宏观指标速查
- **PMI > 50**：经济扩张，利好成长/周期
- **PMI < 50**：经济收缩，利好防御/债券
- **CPI上行**：通胀预期，利好商品/资源
- **CPI下行**：通缩风险，利好债券/防御
- **社融超预期**：流动性宽裕，利好股市
- **美联储加息**：全球流动性收紧，A股承压

## 三、板块轮动框架

### 轮动规律
- **经济上行期**：周期（有色→煤炭→化工）→ 消费 → 成长
- **经济下行期**：成长（超跌反弹）→ 防御（银行→红利）→ 债券
- **政策驱动**：政策落地前炒预期，落地后兑现（利好出尽）

### 资金流向信号
- 连续3日主力净流入 → 趋势确认，可跟进
- 单日巨额流入但板块涨幅不大 → 可能在吸筹
- 巨额流出但板块微涨 → 可能是散户接盘，危险

### 板块强弱判断
| 强势信号 | 弱势信号 |
|----------|----------|
| 板块内多只个股涨停 | 板块内多只个股跌停 |
| 龙头股创新高 | 龙头股破位 |
| 板块指数站上20日均线 | 板块指数跌破20日均线 |

## 四、技术分析框架

### 关键价位
- **支撑位**：前期低点、均线支撑（5/10/20日）、整数关口
- **压力位**：前期高点、均线压制、成交密集区
- **止损位**：买入价下方3-5%，或关键支撑位下方

### 买入信号
1. 回踩支撑位企稳（缩量止跌）
2. MACD金叉（DIF上穿DEA）
3. 均线多头排列（5>10>20>60）
4. 突破关键压力位放量

### 卖出信号
1. 触及压力位滞涨（放量不过）
2. MACD死叉（DIF下穿DEA）
3. 均线空头排列
4. **有效跌破关键支撑位**（见下方「技术止损判定规则」）

### 技术止损判定规则（重要！）

**核心原则：不是所有"跌破均线"都触发止损。必须区分有效跌破和假突破。**

**有效跌破的判定条件（必须同时满足2条以上才触发止损）：**
1. **新鲜跌破**：之前N个交易日在均线上方运行，今日首次跌破。如果已经在均线下方运行多日，不视为"跌破"信号
2. **实体跌破**：收盘价跌破均线幅度 ≥ 1-2%，不是微小刺穿。盘中短暂跌破但收盘收回不算
3. **放量确认**：跌破当日成交量 ≥ 20日均量的1.2倍以上。缩量跌破大概率是假突破
4. **连续确认**：连续2日收盘价在均线下方，比单日跌破更可靠

**无效跌破（不触发止损）：**
- ❌ 已经在均线下方运行多日（不是新鲜跌破，价格已消化）
- ❌ 盘中刺穿但收盘收回均线上方
- ❌ 缩量跌破（量能不足，主力未出逃）
- ❌ 单日跌破但次日立刻收回
- ❌ 市场整体恐慌导致的普跌（非个股/板块问题）
- ❌ 跌破幅度 < 1%的微小刺穿

**操作指引：**
| 信号 | 动作 | 仓位 |
|------|------|------|
| 有效跌破20日均线（多条件共振：新鲜+实体+放量） | 执行止损 | 清仓或减半 |
| 微小刺穿/缩量跌破 | 观察，不急着动 | 持有 |
| 已在均线下方运行多日 | 不视为新止损信号 | 持有，等反弹或另行判断 |
| 跌破后3日内收回均线 | 确认假突破 | 持有 |

## 五、仓位管理框架

### 总仓位控制
| 市场环境 | 建议总仓位 | 说明 |
|----------|------------|------|
| 单边上涨 | 70-90% | 趋势明确，重仓 |
| 震荡市 | 40-60% | 灵活应对 |
| 单边下跌 | 20-40% | 防守为主 |
| 极端恐慌 | 10-20% | 留足子弹抄底 |

### 单只ETF仓位
- **上限**：60%（绝对上限）
- **常规**：10-15%（分散风险）
- **试探**：5%（不确定性高时）

### 加仓节奏
1. 首次建仓：5-10%
2. 确认趋势后加仓：加至15-20%
3. 突破关键位加仓：加至≤60%
4. **永远不在亏损时加仓**（除非有明确左侧信号）

### 止损止盈（2026-06-29更新）

**核心原则：不设固定止盈，让利润奔跑。止损按单笔买入计算，盈利后动态保护。**

- **止损**：单笔买入后亏损≥3%减半仓，≥4%清仓
- **盈利保护**：单个标的盈利超过10%后，止损线上移至3%盈利位置（即盈利回撤到3%时止损保本）
- **止盈**：**不设固定止盈线**，让利润奔跑。仅当单个标的仓位超过60%后减仓
- **其他加减仓**：主要看市场风向及逆向思维，灵活决策
- **技术止损**：跌破关键支撑位（如20日均线）止损，不等比例触发
- **时间止损**：买入后3-5天无盈利且趋势走弱，考虑离场

### 止损止盈计算方法
**按单笔买入计算：**
```
单笔亏损 = 买入价 - 当前价
止损触发 = 单笔亏损 / 买入价 ≥ 3%（减半仓）或 ≥ 4%（清仓）

盈利保护：
单笔盈利 = 当前价 - 买入价
当单笔盈利 ≥ 10% → 止损线上移至：买入价 × (1 + 3%)
即盈利回撤到3%时止损保本

示例：
买入价1.50元 → 跌至1.455元触发止损（亏3%）
买入价1.50元 → 涨至1.65元（盈利10%）→ 止损线上移至1.545元（盈利3%位置）
```

## 六、市场心理框架

### 散户常见错误
1. **追涨杀跌**：看到涨了才买，跌了就割肉
2. **频繁交易**：每天都要操作，手续费吃掉利润
3. **重仓单只**：梭哈一只ETF，风险集中
4. **死扛不止损**：亏损了就放着，期望回本
5. **听消息炒股**：看到新闻就冲进去，成为接盘侠

### 机构行为特征
- 机构建仓：缓慢吸筹，股价波动小，成交量温和放大
- 机构出货：高位震荡，成交量异常放大，利好频出
- 机构洗盘：快速下跌清洗浮筹，然后拉升

### 情绪指标
| 极度恐惧 | 正常 | 极度贪婪 |
|----------|------|----------|
| VIX > 30 | VIX 15-25 | VIX < 12 |
| 新闻全是利空 | 新闻中性 | 新闻全是利好 |
| 周围没人谈股票 | 偶尔有人问 | 人人都是股神 |

## 七-B、竞赛模式三维决策框架

> 竞赛期间（2026/6/11-7/20），策略员出策略时必须使用此框架。核心原则：**外盘定方向，仓位定力度，逆向信号能翻牌。**

### 决策维度一：外盘信号（挡位杆）

外盘信号决定今天的大方向——加仓、退场、还是等待。但不是机械执行，要结合维度二和维度三。

| 外盘信号 | 典型特征 |
|----------|---------|
| **大涨/利好** | 美股三大指数涨>1%，VIX回落，A50期货上涨，无重大利空 |
| **平稳/多空交织** | 美股涨跌<1%，无明确方向，A50期货持平 |
| **大跌/明显利空** | 美股跌>1.5%，VIX飙升>20，地缘冲突/加息/关税等 |

### 决策维度二：仓位现状（力度调节器）

同一外盘信号，仓位不同，动作完全不同：

| | 仓位轻(≤40%) | 仓位适中(40-70%) | 仓位重(>70%) |
|---|---|---|---|
| **大涨/利好** | 加仓，优先加已验证标的 | 持有为主，等回调再加，不追涨 | 不加，考虑部分止盈（利好兑现） |
| **平稳/多空交织** | 小仓位试探进攻型标的 | 看个股：有独立逻辑→加；纯跟随→等 | 不动，等方向明确 |
| **大跌/明显利空** | 逆向：左侧试探建仓(5-10%) | 先止损触及线的标的，保留子弹 | 减仓控风险，但不恐慌全割 |

### 决策维度三：逆向信号（翻牌器）

逆向信号可以推翻维度一和维度二的结论：

| 逆向信号 | 外盘说"加"时 | 外盘说"退"时 | 外盘说"等"时 |
|----------|------------|------------|------------|
| **RSI>80 极度超买** | 以逆向为准：不加，甚至减 | 正常减仓 | 减仓 |
| **RSI<20 极度超卖** | 持仓不动 | 逆向：小仓位左侧试买 | 试探建仓 |
| **VIX>30 极度恐惧** | 保持警惕 | 逆向：准备抄底弹药 | 逆向：逐步建仓 |
| **VIX<12 极度贪婪** | 以逆向为准：收紧 | 正常减 | 减仓 |

### 持仓质量判断（维度二的补充）

不是所有持仓都一样对待：
- **有独立逻辑的持仓**（国产替代催化、政策利好、板块资金连续流入、技术面多头排列）→ 大盘跌也先扛，甚至加仓
- **纯跟随大盘的持仓**（无独立催化、只是跟着指数涨跌）→ 大盘转弱时优先换掉
- **已浮亏且无独立逻辑** → 果断止损，不扛

### 综合决策示例

**示例1：外盘大涨 + 仓位重(75%) + RSI=75**
- 维度一（外盘）：加仓
- 维度二（仓位重）：不加
- 维度三（RSI偏高）：警惕
- **最终决策：持有不动，部分止盈RSI>75的标的**

**示例2：外盘大跌 + 仓位轻(30%) + RSI=25**
- 维度一（外盘）：退
- 维度二（仓位轻）：左侧机会
- 维度三（RSI超卖）：逆向买入
- **最终决策：左侧试探建仓5-10%，选有独立逻辑的超跌标的**

**示例3：外盘平稳 + 仓位适中(55%) + 持仓有独立逻辑**
- 维度一（外盘）：看个股
- 维度二（仓位适中）：有逻辑的加
- 维度三（无极端信号）：正常
- **最终决策：持有并小幅加仓有独立逻辑的标的**

### 选标原则（竞赛模式）

- 优先高beta进攻型板块：半导体、芯片、AI、创新药、军工、机器人、新能源
- **全行业扫描**：不能只看芯片和证券，必须扫描机器人、新能源、创新药、AI、军工等所有进攻型板块的资金流向
- 少碰红利防御类（跑不赢排名）
- 选有独立逻辑的标的（政策催化、资金持续流入、板块轮动确认）
- **持仓≤4只**，集中火力
- ETF选择标准：规模>10亿、日均成交额>5000万

## 八、ETF轮动框架

### 轮动策略类型
1. **动量轮动**：买入近期涨幅最大的ETF，定期轮换
2. **均值回归**：买入近期跌幅最大的ETF，等待反弹
3. **事件驱动**：根据政策/事件催化选择板块
4. **周期轮动**：根据经济周期切换板块（参考美林时钟）

### ETF选择标准
| 标准 | 要求 |
|------|------|
| 规模 | > 10亿（避免流动性风险） |
| 日均成交额 | > 5000万（便于进出） |
| 跟踪误差 | < 2%（紧密跟踪指数） |
| 费率 | 管理费+托管费 < 1% |

### 当前市场适用策略
- **外盘大涨+风险偏好上升** → 动量轮动（跟进强势板块）
- **市场低迷+超跌** → 均值回归（布局超跌板块）
- **政策催化** → 事件驱动（跟随政策方向）
- **经济周期切换** → 周期轮动（美林时钟）

## 十、实战交易经验与避坑指南

> 本章总结A股ETF实战交易中的高频失误和宝贵经验，是策略员日常操作的避坑手册。**每一条都是血泪教训，请逐条对照自检。**

### 1. 开盘前必做功课（30分钟定生死）

**早盘准备清单：**
1. **隔夜外盘复盘**：美股三大指数涨跌幅、VIX指数、A50期货夜盘表现（6:30-7:00更新）
2. **重大新闻扫描**：新华社/证监会/央行/国务院是否有重大政策发布
3. **目标标的复盘**：昨晚策略中的标的，今日开盘价与昨日收盘价的偏差
4. **集合竞价观察**：9:15-9:25期间关注目标ETF的集合竞价量价，判断多空力量

**致命错误：** 不看盘直接开盘就下单 → 你永远不知道昨晚发生了什么。

### 2. 开盘30分钟的黄金法则

**核心规律：A股开盘30分钟的走势，80%情况下预示全天方向。**

| 开盘表现 | 通常含义 | 操作建议 |
|----------|---------|---------|
| 高开后快速回落翻绿 | 多头陷阱，主力出货 | 不追高，等企稳 |
| 低开后快速拉升翻红 | 恐慌洗盘完毕，空头衰竭 | 关注是否放量 |
| 平开后震荡缩量 | 多空分歧不大，等方向 | 不急操作 |
| 高开高走放量 | 强势信号，可跟进 | 等回踩5日均线确认 |

**铁律：开盘前15分钟不交易！** 这段时间波动最大、信息最不充分，冲动交易亏损概率极高。

### 3. 十大高频失误（血泪教训）

**① 追涨杀跌（散户第一大杀手）**
- 症状：看到ETF涨了3%就冲进去买，第二天跌了2%就割肉
- 解药：严格执行「涨幅超3%不追高」原则，等回踩支撑位再进
- 记住：错过一次机会不会死，追错一次高可能亏10%

**② 频繁交易（手续费吞利润）**
- 症状：每天操作5-10笔，手续费累积可观
- 解药：每日操作≤5笔，非必要不动手
- 数据：假设每日交易2笔ETF，一年约500笔，手续费+印花税约0.5%，年化吃掉利润

**③ 满仓梭哈（没有退路）**
- 症状：把所有资金压在一只ETF上
- 解药：单只ETF仓位≤60%，总仓位≤95%（至少留5%现金应对突变）
- 教训：满仓一只，一旦黑天鹅，直接出局

**④ 死扛不止损（散户第二大杀手）**
- 症状：亏了5%不走，亏了10%更不走，期望"回本"
- 解药：单笔亏损≥3%减半仓，≥4%无条件清仓
- 记住：止损是保护本金的盾牌，不是认输

**⑤ 追消息炒股（接盘侠专属）**
- 症状：看到新闻"XXX板块大涨"就冲进去
- 解药：消息出来时，聪明钱早就在里面了，你进去就是接盘
- 原则：利好出尽是利空，利空出尽是利好

**⑥ 盈利不走、亏损死扛（人性弱点）**
- 症状：赚了3%觉得还能赚，结果回吐；亏了5%觉得能反弹，结果亏更多
- 解药：严格执行阶梯止盈止损（见第五章）

**⑦ 忽视成交量（量价背离是陷阱）**
- 症状：只看价格不看量，追涨时量缩价涨（诱多）
- 解药：上涨必须放量确认，缩量上涨是假突破；下跌缩量可以不慌，下跌放量要警惕

**⑧ 过度分散（精力分散=收益分散）**
- 症状：同时持有8-10只ETF，每只仓位5%
- 解药：竞赛模式≤4只集中火力，常规模式≤6只
- 原则：少即是多，集中才能有超额收益

**⑨ 情绪化交易（亏了想翻本，赚了想更多）**
- 症状：亏损后加大仓位想"一把赚回来"，盈利后不收手
- 解药：用三维决策框架替代情绪决策
- 自检：下单前问自己——"这是理性判断还是情绪驱动？"

**⑩ 不复盘（重复犯同样的错）**
- 症状：每天只关注赚亏，不分析原因
- 解药：每日收盘后花15分钟复盘——今天做对了什么？做错了什么？明天怎么改进？
- 记住：不复盘的交易者，永远在同一个坑里跌倒

### 4. ETF交易的特殊陷阱

**① T+1规则陷阱**
- A股ETF是T+1交易，今天买的明天才能卖
- 陷阱：盘中追高被套，当天无法止损，只能等明天
- 对策：**永远不要在尾盘追高**，如果要追，至少在下午2:30前评估走势

**② 溢价/折价陷阱**
- ETF二级市场价格可能偏离净值
- 陷阱：高溢价买入，次日溢价收敛，净值没跌但你亏了
- 对策：关注IOPV（参考净值），溢价>1%的ETF谨慎追入

**③ 流动性陷阱**
- 小规模ETF买卖价差大，进出成本高
- 陷阱：你以为赚了0.5%，实际买卖价差就吃掉了0.3%
- 对策：只选规模>10亿、日均成交额>5000万的ETF

**④ 跟踪误差陷阱**
- 部分ETF跟踪误差大，实际表现与指数偏离
- 陷阱：你以为买的是半导体指数，实际跑输指数2%
- 对策：选择跟踪误差<2%的ETF，优先选头部基金公司的产品

**⑤ 节假日陷阱**
- 长假前（春节、国庆），市场可能大幅波动或缩量
- 陷阱：满仓过节，假期间海外市场暴跌，节后直接低开
- 对策：长假前减仓至50%以下，持有现金过节

### 5. 竞赛模式特殊经验

**① 起跑阶段（第1-7天）**
- 目标：不要大幅落后，建立安全垫
- 策略：偏稳健，仓位40-60%，选确定性高的标的
- 禁忌：第一天就梭哈博高收益

**② 中段追赶（第8-21天）**
- 目标：根据排名差距调整激进程度
- 策略：排名靠前→稳健守住；排名靠后→适当提升beta
- 关键：找到有独立逻辑的标的（政策催化、资金持续流入）

**③ 冲刺阶段（第22-40天）**
- 目标：最后冲刺，收益最大化
- 策略：可适度激进，但不能失控
- 关键：不要在最后几天犯低级错误（追高、满仓）

**④ 排名策略**
- 落后时：不要一次性梭哈翻盘，容易崩盘
- 领先时：不要过度保守，被反超更伤士气
- 关键：每天进步一点点，复利的力量比暴利更可靠

### 6. 市场阶段识别与应对

| 市场阶段 | 典型特征 | 应对策略 |
|----------|---------|---------|
| **底部震荡** | 成交量极度萎缩，利空不跌，RPS弱势 | 分批左侧建仓，耐心等突破 |
| **主升浪** | 放量突破，板块轮动，赚钱效应明显 | 跟随趋势，重仓进攻型ETF |

| **反弹初期** | 缩量企稳，超跌反弹，题材试盘 | 小仓位试探，确认后再加仓 |

| | 盈利时 | 亏损时 |
|---|--------|--------|

### 收益率计算公式（重要！）
**正确公式：**
```
收益率 = (今日总资产 - 昨日总资产) ÷ 昨日总资产 × 100%
```

**示例：**
```
昨日总资产: ¥1,015,536.64
今日总资产: ¥1,024,255.54
当日盈亏: +¥8,718.90

收益率 = 8,718.90 ÷ 1,015,536.64 × 100%
       = +0.86% （正确）
```

**常见错误：**
```
❌ 错误：8,718.90 ÷ 1,024,255.54 ≈ 0.85% （用今日总资产做基数）
❌ 错误：8,718.90 ÷ 1,015,536.64 × 100% ≈ 0.86% （但写成 +2.18%）
```

**规则：**
1. 收益率必须用「昨日总资产」作为基数
2. 计算结果保留两位小数
3. 必须明确标注「昨日总资产」和「今日总资产」

**巡检员/策略员检查清单：**
```
[ ] 收益率计算是否正确？
[ ] 基数是否为「昨日总资产」？
[ ] 结果是否与实际盈亏匹配？
```

## 十二、仓位管理深度技巧（实战补充）

### 以损定仓法（核心）

**先决定"这次最多亏多少"，再反推仓位大小：**

```
仓位大小 = 最大可承受亏损金额 ÷ 止损幅度

示例：账户1000万，单笔最大亏损2%（20万），止损幅度5%
→ 仓位 = 20万 ÷ 5% = 400万（占总资金40%）
```

### 三大控管原则

1. **单笔风险 ≤ 总资金2%**：亏损超过临界值，翻身难度指数级上升（亏50%需赚100%回本）
2. **当日总亏损 ≤ 总资金6%**：含已实现+未实现，达到后不再开新单
3. **以损定仓**：先定最大亏损额，再反推仓位

### ETF仓位管理要点

- 单一行业ETF仓位 ≤ 总资金20%
- 分 **3-5批** 分批布局，绝不一把梭哈
- **永远不要满仓，永远不要空仓**（对中长线投资者而言）
- 空仓（买国债ETF）是规避系统性风险的关键

## 十四、ETF交易特殊陷阱（深度补充）

### T+1规则陷阱
- 买入当天不能卖出，次日才能卖出
- **后果**：如果当天追高买入后当天就跌，只能干瞪眼
- **对策**：买入前确认当日不会出现利空（如重要数据发布、政策落地）

### 溢价/折价陷阱
- 市场情绪狂热时，ETF市价可能大幅高于净值（高溢价）
- **后果**：溢价3%买入，即使指数不涨也亏3%
- **对策**：交易前必查溢价率，普通ETF溢价超3%谨慎，超5%放弃

### 流动性陷阱
- 成交量小的ETF买卖价差大
- **后果**：想卖时卖不出好价钱，想买时买不到好价钱
- **对策**：日均成交额 < 5000万的ETF谨慎操作

### 跟踪误差陷阱
- ETF实际涨跌与跟踪指数涨跌不一致
- **原因**：管理费、跟踪偏差、成分股调整等
- **对策**：选择跟踪误差 < 2%的ETF

## 十五、心态管理（实战补充）

### 心态四象限

| 心态 | 特征 | 应对 |
|------|------|------|
| **亢奋** | 连续盈利，觉得自己是股神 | 最危险的时刻！主动减仓，锁定利润 |
| **恐慌** | 连续亏损，不敢操作 | 最好的机会！冷静分析，逆向布局 |
| **贪婪** | 仓位已重，还想加 | 问自己：如果明天跌10%，能承受吗？ |
| **后悔** | 卖飞了/没买到 | 不要因为后悔而追高或恐慌接刀 |

### 五条铁律

1. **亏损时不加仓摊薄成本**（除非有明确左侧信号）
2. **盈利时让利润奔跑，设好盈利保护**（单笔盈利≥10%后止损上移至3%盈利）
3. **到了止损位必须执行，纪律 > 技术**
4. **不因为一次交易成功就改变策略（幸存者偏差）**
5. **复盘比交易重要——每周回顾一次交易记录**

### 观察池操作规则
1. **加入条件**：板块连续3日资金净流入，或出现明显政策/事件催化
2. **建仓条件**：回调至支撑位 + 资金流向确认 + 总仓位允许
3. **移除条件**：板块连续3日资金净流出，或基本面逻辑破坏
4. **观察频率**：每日晨间更新，盘中巡检跟踪

### 观察池与持仓的关系
- 观察池标的不占用持仓仓位
- 观察池标的可在任何时间建仓（不受"持有/观望"指令限制）
- 建仓后从观察池移除，纳入正常持仓管理
